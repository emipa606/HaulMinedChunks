# Changelog for HaulMinedChunks

1.5.2 - Added support for chunks dug up by Vanilla Framework Expanded-based modded creatures (Stonedigger Dryads, etc) via delmain


1.5.1 - Should now work for colonists, prisoners, mechs and slaves


1.5.0 - Mod updated for 1.5 and passed autotests


1.4.1 - Mod updated for 1.4 and passed autotests.
Used DeepL to update translations for ChineseSimplified, French, German, Russian


1.3.6 - All pawns of the colony should now trigger the haul-marking instead of only colonists. Should now work with all droid-mods etc.


1.3.5 - Fixed chunk-category inheritance


1.3.4 - Should now mark all chunks instead of just stone-chunks. Alpha biomes had metal chunks spawning from some mountain.


1.3.3 - Support for drone-mining in [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2037491557]Project RimFactory - Drones[/url]


1.3.2 - Added settings for auto-marking only in specific areas, also support for [url=https://steamcommunity.com/sharedfiles/filedetails/?id=1776143665]Raise The Roof[/url]


1.3.1 - Should now only mark chunks mined by colonists


1.3.0 - First publish
